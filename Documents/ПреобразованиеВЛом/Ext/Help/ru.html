Документ "Комплектация товаров"

Документ предназначен для отражения операций комплектации лома, изделий и разукомплектации изделий.

Комплектация лома

Комплектация товаров

Разукомплектация товаров

При вводе документа в шапке обязательно нужно указать следующие реквизиты:

  • Организация - организация, по которой осуществляется комплектация или разукомплектация. 
  • Склад - склад, на котором осуществляется комплектация или разукомплектация. 
  • Номенклатуру,  Количество, Вес, Вставки, Размер, Серию номенклатуры и Цену за штуку комплекта.  

Комплектация лома

Для отражения комплектации необходимо выбрать операцию Комплектация лома.

После выбора операции нужно заполнить данные о комплекте в шапке документа и данные о комплектующих в табличной части Комплектующие.

В шапке указывается информация о комплетке (Вид номенклатуры "металл")

В табличной части Комплектующие указываются комплектующие, их количество, вес и другие характеристики.

Данные в таблице комплектующих можно заполнить автоматически, используя конпки из меню Подбор

Комплектация товаров

Для отражения комплектации необходимо выбрать операцию Комплектация товаров.

После выбора операции нужно заполнить данные о комплекте в шапке документа и данные о комплектующих в табличной части Комплектующие.

В шапке указывается информация о комплетке (Вид номенклатуры "ювелирное изделие", "товар")

В табличной части Комплектующие указываются комплектующие, их количество, вес и другие характеристики.

Данные в таблице комплектующих можно заполнить автоматически, используя конпки из меню Подбор

Разукомплектация товаров

Для отражения разукомплектации необходимо выбрать операцию Разукомплектация товаров.

После выбора операции нужно заполнить данные о комплекте в шапке документа используя конпки из меню Подбор комплекта.

В табличной части Комплектующие номенклатура, количество, вес и другие характеристики заполняются автоматически по данным комплекта.

Ручное изменение данных в табличной части Комплектующие запрещено.

Печатные формы

Для документа Комплектация товаров предусмотрены следующие печатные формы:

  • Комплектация номенклатуры / Разукомплектация номенклатуры (в зависимости от выбранного вида операции)
  • Печать этикеток и ценников
  • ТОРГ-16

 

!!! Важно

Комплектация возможно только для собственных товаров и металлов.

Для любых операций с комплектующими (продажа, возврат и т.п.) нужно сделать документ с видом операции Разукомплектация товаров.

Документ с видом операции Разукомплектация товаров рекомендуется делать по организации и складу из документа комплектация.

Данные в таблице комплектующих можно заполнить автоматически, используя конпки из меню Подбор
